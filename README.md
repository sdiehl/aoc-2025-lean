# Advent of Code 2025

[![CI](https://github.com/sdiehl/aoc-2025-lean/actions/workflows/ci.yml/badge.svg)](https://github.com/sdiehl/aoc-2025-lean/actions/workflows/ci.yml)

Lean 4 solutions by [Stephen Diehl](https://adventofcode.com/2025/leaderboard/private/view/5342262?view_key=947bb578).

```bash
lake build
lake test # run all days

.lake/build/bin/aoc2025 1    # run day 1
.lake/build/bin/aoc2025 2    # run day 2
```

Note that Day 10 requires the [HiGHs solver](https://highs.dev/) to be installed:

```bash
brew install highs
```
